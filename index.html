M3U Playlist API

This feature allows you to generate M3U playlists filtered by country. The API is accessible via the following format:

https://opentivu.netlify.app/api/[country_code]

Example supported country codes:

  • it: Italy
  • de: Germany
  • fr: France
  • es: Spain
  • us: USA

If you visit the URL https://opentivu.netlify.app/api/help you will see the list of all countries available.

Examples:

  • For Italian channels: https://opentivu.netlify.app/api/it
  • For German channels: https://opentivu.netlify.app/api/de
